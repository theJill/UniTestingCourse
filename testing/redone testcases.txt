Компания: https://www.aviasales.com


1. 
Кейс: Сделать поиск билета с указанием даты из прошлого.

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
Ввести в поле Depart значение 1 october 2014. Ввести значение From 
"Los Angeles", To "Barcelona". Графу Passangers оставить по умолчанию. 
Поле Return оставить пустым.

Ожидаемый результат: ввод данных с клавиатуры заблокирован. Поверх поля
Depart появляется UI, допускающий ввод данных только путем нажатия 
соответствующей даты в нем.
Даты, находящиеся в прошлом относительно локального 
времени города графы "From" заблокированы для нажатия. Ввод невалидных 
данных, в том числе 1 october 2014 невозможен.

2. 
Кейс: Указать время Depart позже чем Return.

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
В графу Depart ввести значение 18 October 2019, в графу Return ввести 
значение 16 October 2019. Значение From "Los Angeles", To "Barcelona". 
Графу Passangers оставить по умолчанию.  

Ожидаемый результат: после ввода даты 18 October 2019  графу Depart, 
значения, находящиеся ранее 18 October 2019, в том числе 16 October 2019
заблокированы.

Повторить действия ввода в обратном порядке. 

Действия:ввести значение From "Los Angeles", To "Barcelona". Графу 
Passangers оставить по умолчанию. Нажать на поле Return. Ожидается
появление UI. В поле Return ввести значение 16 October 2019.
Нажать на поле Depart. Ожидается появление UI. 
В поле depart ввести значение 18 October 2019.

Ожидаемый результат: открытие UI поля Return. Значение, введенное в поле
Return ранее становится пустой строкой.

3. 
Кейс: Указать время отправления больше чем через год от текущего.

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
Ввести значения From "Los Angeles", To "Barcelona"
Нажать на поле from. Дждаться появления UI. В графу Depart ввести
значение 20 october 2020.

Ожидаемый результат: элемент ввода данной даты заблокирован. 

4. 
Кейс: Установить количество взрослых, детей, младенцев 0 человек.

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
Ввести значения From "Los Angeles", To "Barcelona", depart 1 November 2019.
Нажать на поле Passangers&class. Дождаться появления UI. Ввести значение Adalt = 0 Children = 0 
Infants = 0. Путем нажатия на UI элемент "минус", установить все значения в 0.
Класс оставить по умолчанию.

Ожидаемый результат: UI элемент кнопка становится неактивным при достижении 
значения 0 в полях Cildren, Infants и при достижении значения 1 в поле Adults.

5. 
Кейс: Проверить что на одного взрослого приходится не более 1 младенца на 
этапе поиска и бронирования. 

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
Ввести значения From "Los Angeles", To "Barcelona", depart 1 November 2019.
Нажать на графу Passangers&class. В появившимся UI значение Adalt = 1
оставить по умолчани. Нажать на элемент + возле графы Infants два раза.

Ожидаемый результат: после первого нажатия значение 1, после второго
UI элемент "+" для infants заблокирован, появляется сообщение ошибки 
"Maximum 1 lap infant per adult passenger. If you have more infants, 
add them as children under 12".

8. 
Кейс: Найти билет в поиске и перейти на сайт бронирования.

Действия:
Перейти на главную страницу сайта по ссылке https://www.aviasales.com
Ввести значения From "Los Angeles", To "Barcelona", depart 1 November 2019.
Нажать кнопку "Search flights". 
В центральной части экрана выбрать верхний <div class="product-list__item fade-enter-done">
(прямоугольный блок с описанием билета), нажать кнопку book.

PLEASE HOLD ON A FEW SECONDS…"
